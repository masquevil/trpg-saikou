## 包含工具

### 车卡工具

https://masquevil.github.io/trpg-saikou/#/coc-card

![截图](./public/coc-card-eg.png)

1. 生成正反面图片，方便打印
2. 可以打印空白卡线下车卡
3. 也可以在线车卡

### 记录工具

![example](./public/example.png)

1. 需拉取本仓库代码后本地使用
2. 记录自己 TRPG 打卡记录、评价、想玩程度
3. 方便截图分享寻找队友拼车

## Project Setup

```sh
yarn install

sh ./shell/copy.sh

yarn dev
```
